COA: API Documentation

Documentation linked here is intended for the use of software developers to use and extend COA packages. Except where specifically stated otherwise, the implementation language is C++.
This documentation focuses on the SUP (Supervision and Automation System) and PSPS (Pulse Schedule Preparation System).
